Usage

A small function that takes a valid color parameter and returns either a black or white color, whichever will be the most legible. A global white or black value is declared if a particular theme does not have one of its own.

$white: #fff !default;
$black: #101010 !default;

@function high-contrast-color($color) {
  // Is this color light or darK?
  $color-lod: light-or-dark($color);

  @if ($color-lod == "dark") {
    @return $white;
  } @else {
    @return $black;
  }
}
// Usage:
.example {
  $color: #0094c2;
  background-color: $color;
  color: high-contrast-color($color)
}
// Output:
.example {
  background-color: #0094c2;
  color: #101010;
}

This function is located in the a11y-color.scss file along with the a11y-color() functions.
