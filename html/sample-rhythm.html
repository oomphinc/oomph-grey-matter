The line-height of the grid should be adjusted for each project, as well as the padding-top to align to the first element. This site has a 18px font size and a 1.4 line height, which results in a pixel line-height of 25px. Therefore, the CSS looks like this:

background: linear-gradient(#c0c 0px, #fff 1px) repeat;
background-size: 100% 25px;
padding-top: 6px;

Adjust the '25' in the background-size line to change the space between lines. Look at the rendered line-height on a <p> to see what the value needs to be. The adjust the padding-top offset if needed.
